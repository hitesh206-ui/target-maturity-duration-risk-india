## Project Objective

This repository supports a public-data research project measuring duration risk, yield compensation, and interest-rate sensitivity in Indian target maturity debt funds.

## Core Research Question

Do Indian target maturity debt funds expose investors to hidden duration risk that is not fully understood from the product label or maturity date alone?

## Research Focus

The project studies:

- modified duration,
- Macaulay duration,
- average maturity,
- yield to maturity,
- YTM-to-duration compensation,
- estimated NAV impact under interest-rate shocks,
- maturity-bucket differences across funds,
- disclosure completeness.

## Planned Data Sources

The project uses public sources only:

- AMC monthly factsheets,
- AMC scheme pages,
- AMFI portfolio disclosure portal,
- public mutual fund disclosure documents,
- scheme benchmark information.

## Planned Methodology

The project will estimate duration risk using:

- modified duration,
- stress testing under interest-rate shocks,
- maturity-bucket comparison,
- YTM-to-duration ratios,
- disclosure-quality analysis.

- duration-risk rankings,
